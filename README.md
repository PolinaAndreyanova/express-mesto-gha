# Бэкенд проекта Место

**Описание**

Бэкенд часть проекта [Место](https://github.com/PolinaAndreyanova/react-mesto-auth)

**Функциональность**
* Авторизация и регистрация пользователя
* Редактирование профиля

* Лайк карточек
* Удаление карточек

**Стек технологий**
* JavaScript
* Node.js
* MongoDB
